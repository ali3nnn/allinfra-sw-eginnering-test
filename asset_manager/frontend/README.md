## Prerequisites

1. Install ```NodeJS```.
2. Install  ```npm```.

## Running the app

1. Go to ```/asset_manager/frontend``` and run ```npm install```.
2. Create the file ```/asset_manager/frontend/src/.env``` and add ```REACT_APP_BASE_URL``` as the API host.
3. Run ```npm run start```.
